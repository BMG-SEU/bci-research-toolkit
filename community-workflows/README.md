# 社区工作流市场

把你写的好用工作流分享出来，其他人 `git pull` 就能用。

## 贡献方法

1. 把你的 `.yaml` 文件放到这个目录
2. 发 Pull Request
3. 在 PR 描述里写清楚：用途、输入参数、适用场景

## 命名规范

```
用途-方法-版本.yaml
例: meg-resting-connectivity-v1.yaml
例: eeg-motor-imagery-csp-lda-v1.yaml
```
